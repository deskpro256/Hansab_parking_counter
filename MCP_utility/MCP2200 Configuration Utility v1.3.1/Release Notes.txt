Important Information   
-------------------------------------------------------------------

******* Driver Installation *******

  BE SURE TO ACCEPT ANY WARNINGS GIVEN DURING THE INSTALLATION. 
A required driver for the MCP2200 USB-to-Serial device is installed 
during the application installation and the warnings must be 
accepted in order for the application to recognize the device.


****** Driver Uninstallation *******

  To uninstall this driver, simply go to the Windows Control Panel
and select "Add/Remove Programs" (or "Programs and Features" in 
Vista and 7).  Double-click (uninstall) the item named the following:
"Windows Driver Package - Microchip Technology, Inc. (usbser) ..."
